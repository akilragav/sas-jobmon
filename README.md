### Prerequisites

You need the following components:

- SAS Version 9.2 or higher
- SAS DI Server
- SAS Stored Process Web Application (a component of SAS BI Server)
- (IBM) Platform LSF
- A database (Oracle, MS SQL Server, Postgres, MySQL, SAS/SHARE, SAS)
